# Пример: bidirectional streaming клиент

Файл: `bidi-streaming-client.go`

## Назначение
Отправляет несколько сообщений и одновременно читает ответы сервера (эхо). Демонстрирует двунаправленный стриминг.

## Запуск
```bash
go run bidi-streaming-client.go
```
Вы увидите отправленные сообщения и эхо-ответы от сервера.
