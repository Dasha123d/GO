# Rate Limiting

Пример ограничения частоты запросов (Rate Limiter).

## Описание
Используется алгоритм Token Bucket. Сервер ограничивает количество запросов от одного IP-адреса.

## Настройки из кода
- `rate`: 5 запросов в секунду.
- `burst`: 10 (максимальный всплеск запросов).

Если лимит превышен, возвращается статус `429 Too Many Requests`.

## Зависимости
Используется библиотека: `golang.org/x/time/rate`
