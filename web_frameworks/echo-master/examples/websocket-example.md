# WebSocket Support

Пример создания WebSocket соединения для двусторонней связи.

## Описание
Используется библиотека `gorilla/websocket` для апгрейда HTTP соединения до WebSocket протокола.

## Как это работает
1. Клиент подключается к `/ws`.
2. Сервер выполняет `Upgrade`, переключая протокол.
3. Открывается бесконечный цикл: сервер ждет сообщение от клиента и отправляет его обратно ("Эхо").

## Зависимости
Используется библиотека: `github.com/gorilla/websocket`
