# Graceful Shutdown

Пример корректного завершения работы сервера.

## Описание
Сервер обрабатывает сигналы операционной системы (SIGINT, SIGTERM). При получении сигнала (например, Ctrl+C) сервер перестает принимать новые подключения и ждет завершения текущих запросов перед выходом.

## Ключевые моменты
1. `signal.Notify` — подписка на системные сигналы.
2. `e.Shutdown(ctx)` — метод Echo для плавного завершения с таймаутом.
3. Сервер запускается в отдельной горутине `go e.Start()`.
