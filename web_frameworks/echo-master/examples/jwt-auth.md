# JWT Authentication

Пример защиты маршрутов с помощью JWT (JSON Web Tokens).

## Описание
Демонстрирует выдачу токена при логине и проверку этого токена при доступе к защищенным ресурсам.

## Как использовать

1. **Получить токен:**
   Отправьте POST запрос на `/login` с JSON:
   ```json
   { "username": "admin", "password": "password" }
