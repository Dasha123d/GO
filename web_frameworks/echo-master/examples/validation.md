# Data Validation

Пример валидации входящих данных с помощью структурных тегов.

## Описание
Используется библиотека `go-playground/validator`. Данные из запроса автоматически проверяются на соответствие правилам, указанным в тегах структуры.

## Правила валидации (из примера)
- `required`: поле обязательно.
- `min=3`: строка не менее 3 символов.
- `max=20`: строка не более 20 символов.
- `email`: проверка формата email.

## Зависимости
Используется библиотека: `github.com/go-playground/validator/v10`
