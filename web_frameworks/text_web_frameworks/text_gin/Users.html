Пользователи

Gin используется в продакшене компаниями и проектами с открытым исходным кодом любого масштаба. Ниже представлена подборка известных проектов, построенных на Gin. Если ваш проект использует Gin, мы будем рады добавить его в этот список.

Список проектов

  • gorush: Сервер push-уведомлений, написанный на Go.
  • fnproject: Бессерверная платформа, нативная для контейнеров, независимая от облака.
  • photoprism: Фото-приложение на базе ИИ для децентрализованного веба.
  • lura: Сверхпроизводительный API Gateway с middleware.
  • picfit: Сервер изменения размеров изображений, написанный на Go.
  • gotify: Простой сервер для отправки и получения сообщений в реальном времени через WebSocket.
  • cds: Платформа с открытым исходным кодом корпоративного уровня для непрерывной доставки и DevOps-автоматизации.
  • drone: Платформа непрерывной доставки на основе контейнеров, построенная на Go.
  • go-admin: Фреймворк визуализации данных и админ-панели для Go.
  • room: A drop in waiting room middleware for high throughput web applications.

Добавьте свой проект

Используете Gin в продакшене или в проекте с открытым исходным кодом? Откройте pull request в репозитории gin-gonic/website, чтобы добавить ваш проект на эту страницу.
