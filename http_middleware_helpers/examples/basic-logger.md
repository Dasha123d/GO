# Пример: логирующий middleware

Файл: `basic-logger.go`

## Назначение
Показывает создание middleware, который логирует метод, путь, статус-код и время обработки каждого запроса. Используется обёртка `responseRecorder` для захвата статуса.

## Запуск
```bash
go run basic-logger.go
```
При обращении к `http://localhost:8080` в консоль выводится лог.
