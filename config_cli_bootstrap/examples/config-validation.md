# Пример: валидация конфигурации

Файл: `config-validation.go`

## Назначение
Показывает, как проверить корректность значений после загрузки конфигурации. Позволяет отловить недопустимые комбинации на старте.

## Запуск
```bash
go run config-validation.go
```
Завершится с ошибкой, так как `Workers = 0`. Поменяйте значение для успешного запуска.
