# Пример: загрузка из переменных окружения

Файл: `envconfig.go`

## Назначение
Использование библиотеки `envconfig` для автоматического связывания переменных окружения с полями структуры.

## Зависимости
- `github.com/kelseyhightower/envconfig`

## Запуск
```bash
export DB_HOST=localhost
go run envconfig.go
```
Выведет значения, взятые из окружения с учётом значений по умолчанию.
