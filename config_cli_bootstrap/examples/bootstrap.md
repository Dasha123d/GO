# Пример: типичный main() с bootstrap

Файл: `bootstrap.go`

## Назначение
Демонстрирует структурированную начальную загрузку приложения: загрузка конфигурации, инициализация логгера, подключение к БД, запуск HTTP-сервера и корректное завершение по сигналу.

## Зависимости
- `github.com/lib/pq` (драйвер PostgreSQL) или любой другой.

## Подготовка
Укажите корректный DSN в функции `loadConfig()`.

## Запуск
```bash
go run bootstrap.go
```
Сервер стартует на `:8080`. При запросе отображает версию PostgreSQL.
