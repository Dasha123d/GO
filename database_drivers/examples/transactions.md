# Пример: SQL-транзакции

Файл: `transactions.go`

## Назначение
Демонстрация ACID-транзакции с блокировкой строки (`FOR UPDATE`) и откатом при нехватке средств. Пример написан для PostgreSQL с драйвером `pgx`, но легко адаптируется под SQLite (замена драйвера и DSN).

## Зависимости
- `github.com/jackc/pgx/v5` (для PostgreSQL) или `github.com/mattn/go-sqlite3` (для SQLite)

Установите соответствующий драйвер.

## Конфигурация
Для PostgreSQL: измените DSN на свои параметры. Создайте базу `testdb`, если требуется.
Для SQLite: замените драйвер на `"sqlite3"` и DSN на `":memory:"` (уберите `TRUNCATE`, в SQLite он не нужен, просто пересоздайте таблицу).

## Запуск
```bash
go run transactions.go
```

## Ожидаемый вывод (при успехе)
```text
Начальные балансы: Alice=1000.00, Bob=500.00
Переведено 200.00 от Alice к Bob
Итоговые балансы:
  Alice: 800.00
  Bob: 700.00
```

Если изменить сумму на 2000 (недостаточно средств), вывод покажет ошибку и балансы не изменятся.
