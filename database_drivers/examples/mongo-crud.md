# Пример: CRUD с MongoDB

Файл: `mongo-crud.go`

## Назначение
Демонстрация основных операций с MongoDB: подключение, вставка, поиск, обновление и удаление документа.

## Зависимости
- `go.mongodb.org/mongo-driver`

Установите: `go get go.mongodb.org/mongo-driver`

## Требования
Запущенный сервер MongoDB (по умолчанию на `localhost:27017`). Если используете Docker:
```bash
docker run -d --name mongo -p 27017:27017 mongo:7
```

## Запуск
```bash
go run mongo-crud.go
```

## Ожидаемый вывод
```text
Подключено к MongoDB
Вставлен документ с ID: 65f...
Найден пользователь: {ID:ObjectID(...) Name:Alice Age:30}
Обновлено документов: 1
После обновления: возраст=31
Удалено документов: 1
Документ успешно удалён (не найден)
```
