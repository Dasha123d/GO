# Пример: Подключение к PostgreSQL

Файл: `postgres-connect.go`

## Назначение
Демонстрирует базовое подключение к PostgreSQL с помощью драйвера `pgx` через стандартный интерфейс `database/sql`. Выполняется создание таблицы, вставка и чтение записи с использованием контекста и таймаутов.

## Зависимости
- `github.com/jackc/pgx/v5` (stdlib)

Установите: `go get github.com/jackc/pgx/v5`

## Конфигурация
Перед запуском измените DSN в коде на ваши параметры подключения (пользователь, пароль, хост, порт, имя базы).  
По умолчанию: `postgres://postgres:password@localhost:5432/testdb?sslmode=disable`.

Убедитесь, что сервер PostgreSQL запущен и база `testdb` существует (или создайте: `createdb testdb`).

## Запуск
```bash
go run postgres-connect.go
```
## Ожидаемый вывод
```text
Успешное подключение к PostgreSQL!
Добавлена запись с id=1
Получена запись: message="Привет, Go!", created_at=2025-01-01T12:00:00+03:00
```
Идентификатор и время будут конкретными.
