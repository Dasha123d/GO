# Пример: heartbeat (ping/pong) WebSocket

Файл: `heartbeat.go`

## Назначение
Демонстрирует, как настроить автоматические ping-сообщения для поддержания keep-alive и обнаружения обрыва соединения (если pong не получен вовремя, соединение закрывается). Используется `gorilla/websocket`.

## Запуск
```bash
go run heartbeat.go
```
Сервер слушает `:8080/ws`. Если клиент не отвечает на ping в течение 60 секунд, сервер закроет соединение.

## Зависимости
* `github.com/gorilla/websocket`
