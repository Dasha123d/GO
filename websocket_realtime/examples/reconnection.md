# Пример: клиент с переподключением

Файл: `reconnection.go`

## Назначение
Клиент на `nhooyr.io/websocket`, который в случае разрыва соединения автоматически пытается переподключиться с экспоненциальной задержкой (до 30 секунд).

## Запуск
```bash
go run reconnection.go
```
Требуется работающий WebSocket-сервер (например, `echo-server.go`). Клиент будет бесконечно пытаться подключиться и читать сообщения.

## Зависимости
* `nhooyr.io/websocket`
