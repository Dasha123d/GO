# Пример: чат-сервер WebSocket

Файл: `chat-server.go`

## Назначение
Реализует многопользовательский чат, где каждое сообщение от любого клиента рассылается всем подключённым (включая отправителя). Используется паттерн Hub с регистрацией/дерегистрацией клиентов и каналом broadcast.

## Зависимости
- `github.com/gorilla/websocket`

## Запуск
```bash
go run chat-server.go
```
## Проверка
Откройте несколько терминалов и подключитесь:
``bash
wscat -c ws://localhost:8080/ws
```
Сообщения, напечатанные в любом окне, появятся во всех остальных.
