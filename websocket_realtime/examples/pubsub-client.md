# Пример: pub/sub клиент WebSocket

Файл: `pubsub-client.go`

## Назначение
Показывает, как клиент на `nhooyr.io/websocket` может отправлять JSON-сообщения для подписки на канал и получать асинхронные уведомления. Для демонстрации используется публичный echo-сервер, который просто возвращает отправленное — подписка эмулируется.

## Зависимости
- `nhooyr.io/websocket`

## Запуск
```bash
go run pubsub-client.go
```

## Примечание
Чтобы пример работал осмысленно, нужен WebSocket-сервер, реализующий паттерн pub/sub. Код адаптируйте под свой сервер: замените URL и структуру сообщений.
